🏠 House Price Prediction
🔍 Problem
- Predict house prices using regression models.

📂 Dataset
- Boston Housing Dataset

🧠 Steps Performed
- Data Cleaning & EDA
- Correlation analysis
- Model Building (Linear, Ridge, Lasso)
- Model Evaluation using MAE & MSE

🚀 Best Model
- Ridge Regression
  
📊 Key Insight
- Number of rooms positively affects price
- Lower income areas reduce price

  Built a house price prediction model using Linear, Ridge, and Lasso regression,
 achieving low prediction error (MAE ≈ 3.1) and identifying key features affecting housing prices.
